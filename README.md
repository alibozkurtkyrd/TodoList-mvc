# TodoList-mvc
It's a To-Do-List app created by using ASP.NET MVC  models with Entity Framework.


it makes  CRUD web application to manage todo list online.

Database First Approach is used.

---
###  Technologies

* .Net 6
* Entity Framework Core
* Mysql
* c#
* Html
* Css
* Javascript
* Bootstrap
* Visual Studio Code
* Dbeaver
